# Flipper Zero Variometer

Вариометр для Flipper Zero на базе датчика давления BME280.  
Показывает высоту, вертикальную скорость, температуру и давление.  
Подходит для параглайдинга, хайкинга и других активностей на открытом воздухе.

![Flipper Zero Variometer](docs/screenshot.png)

---

## Возможности

- 📡 Поддержка BME280 и BMP280 по I2C
- 🏔️ Отображение высоты (м) в реальном времени
- ↑↓ Вертикальная скорость с фильтром Калмана (м/с)
- 🌡️ Температура (°C) и давление (гПа) на экране
- 🔊 Звуковая индикация: бипы при подъёме, тон при снижении
- 💡 Управление подсветкой
- 📝 Запись данных в CSV файл на SD карту
- 🔄 Автоматическое переподключение датчика при сбое

---

## Подключение датчика BME280

```
Flipper Zero GPIO    BME280
─────────────────────────────
Пин  9  (3.3V)  →   VCC
Пин 11  (GND)   →   GND
Пин 16  (C0)    →   SCL
Пин 15  (C1)    →   SDA
```

> **Адрес I2C** определяется пином SDO датчика:
> - SDO → GND → адрес `0x76` (по умолчанию)
> - SDO → VCC → адрес `0x77`
>
> Приложение автоматически пробует оба адреса.

## Управление

| Кнопка      | Действие                         |
|-------------|----------------------------------|
| ↑ Вверх     | Вкл/выкл звук                    |
| ↓ Вниз      | Вкл/выкл постоянную подсветку    |
| OK          | Открыть меню настроек            |
| Назад       | Закрыть меню / выйти             |

### Меню настроек (кнопка OK)

| Пункт      | Описание                                              |
|------------|-------------------------------------------------------|
| Sound      | Звуковая индикация вертикальной скорости              |
| Backlight  | Постоянная подсветка экрана                           |
| Logging    | Запись данных на SD карту в `/ext/vario_log_DATE.csv` |
| Interval   | Интервал записи: 1, 2, 5, 10, 30 или 60 секунд       |

---

## Звуковая индикация

| Ситуация           | Сигнал                                              |
|--------------------|-----------------------------------------------------|
| Подъём > 0.3 м/с   | Короткие бипы, частота растёт с ростом скорости     |
| Снижение < −0.9 м/с| Низкий тон, частота падает с ростом скорости        |
| Нейтральная зона   | Тишина                                              |

## Сборка

### Требования

- [Unleashed Firmware](https://github.com/DarkFlippers/unleashed-firmware) или официальная прошивка Flipper Zero
- Python 3.x, Git

### Шаги сборки

```bash
# 1. Клонируем firmware (если ещё не сделано)
git clone --recursive https://github.com/DarkFlippers/unleashed-firmware.git
cd unleashed-firmware

# 2. Копируем приложение
cp -r /path/to/flipper-vario applications_user/vario

# 3. Собираем FAP
./fbt fap_dist APPSRC=applications_user/vario

# Готовый файл: build/f7-firmware-D/.extapps/vario.fap
```

### Установка на Flipper Zero

1. Скопируй `vario.fap` на SD карту в папку `/apps/GPIO/`
2. На Flipper Zero: **Apps → GPIO → Vario**

## Технические детали

### Фильтр Калмана

Приложение использует двумерный фильтр Калмана для оценки давления и скорости его изменения. Это даёт плавные показания высоты без задержки:

- **Состояние**: `[давление (Па), скорость изменения давления (Па/с)]`
- **Измерение**: сырое давление с BME280
- `KF_VAR_MEASUREMENT = 0.1` — доверие к датчику
- `KF_VAR_ACCEL = 0.75` — ожидаемое ускорение изменений

### Настройки BME280

| Параметр             | Значение       | Причина                              |
|----------------------|----------------|--------------------------------------|
| IIR фильтр           | 16             | Подавление ветровых импульсов        |
| Standby              | 125 мс         | ~5 обновлений/сек                    |
| Оверсэмплинг темп.   | ×2             | Достаточно для компенсации P         |
| Оверсэмплинг давл.   | ×4             | Баланс точности и скорости           |
| Оверсэмплинг влажн.  | ×1             | Вторичный параметр                   |
