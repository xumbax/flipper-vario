# Flipper Zero Variometer

Вариометр для Flipper Zero на базе датчика давления BME280.  
Показывает высоту, вертикальную скорость, температуру и давление.  
Подходит для параглайдинга, хайкинга и других активностей на открытом воздухе.

---

## Возможности

- 📡 Поддержка BME280 и BMP280 по I2C (автодетект адреса 0x76/0x77)
- 🏔️ Отображение высоты (м) крупным шрифтом
- ↑↓ Вертикальная скорость с фильтром Калмана (м/с)
- 📊 Графический индикатор вертикальной скорости — полоса слева со стрелкой
- 🌡️ Температура (°C) и давление (гПа)
- 🔊 Звуковая индикация: бипы при подъёме, тон при снижении
- 💡 Управление подсветкой
- 📝 Запись данных в CSV файл на SD карту

---

## Подключение датчика BME280

```
Flipper Zero GPIO    BME280
─────────────────────────────
Пин  9  (3.3V)  →   VCC
Пин 11  (GND)   →   GND
Пин 16  (C0)    →   SCL
Пин 15  (C1)    →   SDA
```

> SDO → GND → адрес `0x76` (по умолчанию)  
> SDO → VCC → адрес `0x77`  
> Приложение автоматически пробует оба адреса.

## Управление

| Кнопка  | Основной экран              | В меню настроек           |
|---------|-----------------------------|---------------------------|
| ↑ Вверх | Вкл/выкл звук               | Переход вверх             |
| ↓ Вниз  | Вкл/выкл постоянную подсветку | Переход вниз            |
| OK      | Открыть меню настроек       | Переключить параметр      |
| Назад   | Выйти из приложения         | Закрыть меню              |

### Меню настроек

| Пункт        | Описание                                              |
|--------------|-------------------------------------------------------|
| Sound        | Звуковая индикация вертикальной скорости              |
| Backlight    | Постоянная подсветка экрана                           |
| Logging      | Запись данных на SD карту в `/ext/vario_log_DATE.csv` |
| Log interval | Интервал записи: 1, 2, 5, 10, 30 или 60 секунд       |

## Графический индикатор скорости

Слева на экране — вертикальная полоса 6×56 пикселей:

```
  ╔══╗  ← острие стрелки (подъём 5+ м/с)
  ║██║
  ║██║  ← заливка пропорционально скорости
  ╠══╣  ← нейтраль (центр)
  ║  ║
  ╚══╝
```

- Заливка и стрелка вверх = подъём
- Заливка и стрелка вниз = снижение
- Полная шкала = ±5 м/с

---

## Звуковая индикация

| Ситуация            | Сигнал                                               |
|---------------------|------------------------------------------------------|
| Подъём > 0.3 м/с    | Короткие бипы 1300..2300 Гц, частота растёт          |
| Снижение < −0.9 м/с | Низкий тон 300..1000 Гц, частота падает              |
| Нейтральная зона    | Тишина                                               |

## Сборка

```bash
git clone --recursive https://github.com/DarkFlippers/unleashed-firmware.git
cd unleashed-firmware
cp -r /path/to/flipper-vario applications_user/vario
./fbt fap_dist APPSRC=applications_user/vario
# Результат: build/f7-firmware-D/.extapps/vario.fap
```

Скопируй `vario.fap` на SD карту в `/apps/GPIO/`.
